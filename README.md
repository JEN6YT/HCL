# Getting Started

## Requirement

To install packages: 

```bash
pip install -r requirements.txt
```

## Usage

To run the script in the Python environment:

For us_census data:
```bash
python run_kdd_experiment_us_census.py
```
For covertype data:
```bash
python run_kdd_experiment_covtype.py
```
For ponpare data:
```bash
python run_kdd_experiment_ponpare.py
```
